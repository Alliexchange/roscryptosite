# Регистрация на бирже HTX (Huobi)

HTX (ранее Huobi) — одна из крупнейших криптовалютных бирж мира. В этой статье мы покажем, как создать аккаунт на бирже.

## Шаг 1: Переход на сайт

1. Откройте официальный сайт биржи по нашей реферальной ссылке: [Регистрация на HTX](https://www.htx.com.pk/invite/ru-ru/1f?invite_code=c8tsd223)
2. В правом верхнем углу нажмите кнопку **"Регистрация"**

> **Реферальный код:** `c8tsd223`

> **Важно:** Используйте только официальный сайт биржи. Остерегайтесь фишинговых сайтов!

## Шаг 2: Заполнение данных

Выберите способ регистрации:

### Регистрация по Email

1. Введите ваш email адрес
2. Придумайте надежный пароль (минимум 8 символов, буквы и цифры)
3. Введите код подтверждения с картинки
4. Примите условия использования
5. Нажмите **"Зарегистрироваться"**

### Регистрация по номеру телефона

1. Выберите страну (+7 для России)
2. Введите номер телефона
3. Получите SMS с кодом подтверждения
4. Придумайте пароль
5. Нажмите **"Зарегистрироваться"**

## Шаг 3: Подтверждение email/телефона

После регистрации вам придет:
- **Email:** письмо со ссылкой для подтверждения
- **Телефон:** SMS с кодом

Перейдите по ссылке или введите код для активации аккаунта.

## Шаг 4: Настройка безопасности

Рекомендуем сразу настроить:

1. **Двухфакторная аутентификация (2FA)**
   - Скачайте Google Authenticator
   - Отсканируйте QR-код в настройках безопасности
   - Сохраните резервные коды

2. **Антифишинговый код**
   - Придумайте секретное слово
   - Оно будет отображаться во всех письмах от биржи

## Шаг 5: Верификация (KYC)

Для полного доступа к функциям биржи пройдите верификацию:

1. Перейдите в **Настройки** → **Верификация личности**
2. Выберите **Базовая верификация**
3. Загрузите:
   - Фото паспорта (разворот с фото)
   - Селфи с паспортом в руках
4. Дождитесь проверки (обычно 1-24 часа)

## Готово!

После завершения верификации вы сможете:
- Пополнять счет фиатной валютой
- Покупать и продавать криптовалюту
- Выводить средства
