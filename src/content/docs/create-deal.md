# Как создать сделку

В этой статье мы подробно разберем процесс обмена криптовалюты через наш сервис.

## Покупка USDT

### Шаг 1: Заполните заявку на обмен

1. Нажмите кнопку **"Купить USDT"** на главной странице
2. В открывшемся модальном окне заполните все необходимые данные:
   - **Сумма в USDT** — укажите количество USDT, которое хотите купить
   - **Фамилия** — обязательное поле (будет использовано для проверки платежа)
   - **Имя** — обязательное поле
   - **Отчество** — обязательное поле
   - **Telegram** — ваш username в формате @username (для связи с оператором)
   - **Метод оплаты** — выберите способ оплаты (Сбербанк или Альфабанк)
3. Прочитайте и примите условия сервиса
4. Нажмите **"Создать заявку"**

**⚠️ Важно:** Используйте только свои реквизиты! При оплате с карты третьего лица банк может заморозить возврат до 30 дней.

### Шаг 2: Свяжитесь с оператором

После нажатия кнопки "Создать заявку":
- Автоматически откроется Telegram с готовым сообщением для оператора
- В сообщении будут указаны все введенные вами данные
- Оператор получит вашу заявку и свяжется с вами

### Шаг 3: Получите курс и реквизиты

Оператор ответит вам с:
- Актуальным курсом обмена
- Точной суммой в рублях для оплаты
- Реквизитами для перевода

### Шаг 4: Выполните перевод

1. Переведите указанную сумму на предоставленные реквизиты
2. Сделайте скриншот чека
3. Отправьте скриншот оператору в Telegram

### Шаг 5: Получите USDT

1. Отправьте оператору адрес вашего кошелька
2. Укажите сеть (рекомендуем TRC-20 для низкой комиссии)
3. Дождитесь зачисления (обычно 5-15 минут)

## Продажа USDT

### Шаг 1: Заполните заявку на обмен

1. Нажмите кнопку **"Продать USDT"** на главной странице
2. В открывшемся модальном окне заполните все необходимые данные:
   - Переключитесь на вкладку **"ПРОДАТЬ USDT"** (если она не выбрана по умолчанию)
   - **Сумма в USDT** — укажите количество USDT, которое хотите продать
   - **Фамилия** — обязательное поле
   - **Имя** — обязательное поле
   - **Отчество** — обязательное поле
   - **Telegram** — ваш username в формате @username
   - **Метод оплаты** — выберите способ получения денег (Сбербанк или Альфабанк)
3. Прочитайте и примите условия сервиса
4. Нажмите **"Создать заявку"**

**⚠️ Важно:** Реквизиты для получения денег должны быть оформлены на ваше имя!

### Шаг 2: Свяжитесь с оператором

После создания заявки откроется Telegram с готовым сообщением для оператора.

### Шаг 3: Получите адрес для перевода

Оператор предоставит:
- Курс выкупа
- Сумму в рублях, которую вы получите
- Адрес кошелька для перевода USDT
- Сеть для перевода

### Шаг 4: Переведите USDT

1. Отправьте USDT на указанный адрес
2. Используйте сеть, указанную оператором (обычно TRC-20)
3. Отправьте хеш транзакции (TxID) оператору

### Шаг 5: Получите рубли

1. Укажите реквизиты для получения (номер карты Сбербанка или Альфабанка)
2. Дождитесь перевода (обычно 5-30 минут после подтверждения транзакции)

## Лимиты и комиссии

| Параметр | Значение |
|----------|----------|
| Минимальная сумма | 50 USDT / 5 000 ₽ |
| Максимальная сумма | Без ограничений* |
| Комиссия | Включена в курс |
| Время обработки | 5-30 минут |

*Для крупных сумм свыше 500 000 ₽ курс обсуждается индивидуально.

## Рекомендации

- **Используйте сеть TRC-20** — низкие комиссии и быстрые переводы
- **Проверяйте адреса** — криптовалютные переводы необратимы
- **Сохраняйте переписку** — на случай спорных ситуаций

## Часто задаваемые вопросы

**Что делать, если перевод не дошел?**

Не паникуйте! Отправьте хеш транзакции оператору. Мы проверим статус и поможем решить проблему.

**Можно ли отменить сделку?**

До момента перевода средств — да. После перевода отмена невозможна.
